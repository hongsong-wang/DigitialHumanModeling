# 报告摘要
元宇宙热度减退后，生成式人工智能快速发展。高保真数字人生成和多模态数字人驱动成为AIGC领域最重要的研究问题之一。本报告将介绍数字人技术背景，相关企业应用与学术研究，并介绍汇报人针对数字人生成和数字人多模态驱动两方面的研究工作。
